## Agent skills

### Issue tracker

GitHub Issues, no PR triage. See `docs/agents/issue-tracker.md`.

### Triage labels

All five labels use the default names: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context — `CONTEXT.md` at root, future ADRs in `docs/adr/`. See `docs/agents/domain.md`.
