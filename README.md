# Interactive Comments Section

A responsive comments section built with React, featuring upvoting, replying, editing, and deleting comments.

## Tech Stack

- React 18 + Vite
- CSS Modules
- Vitest + React Testing Library

## Features

- Nested reply threads
- Upvote/downvote scoring
- Edit and delete your own comments
- Real-time relative timestamps
- localStorage persistence
- Responsive (mobile + desktop)

## Getting Started

```bash
pnpm install
pnpm dev
```

## Testing

```bash
pnpm test
```

## Build

```bash
pnpm build
```
